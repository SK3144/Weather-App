## How to Run Locally
1. Clone this repository:
2. Open `index.html` in a browser.

## API Key Setup
1. Get a free API key from [OpenWeather](https://openweathermap.org/api).
2. Replace `YOUR_OPENWEATHER_API_KEY` in `script.js` with your API key.
